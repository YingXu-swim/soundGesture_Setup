Android:

When downloading the .mmp and .pd files, make sure to put both files inside the mobmuplat folder.
You can do so by using your archive app, going to download, selecting the files and the clicking on “Move”.
Alternatively you could also use Android File Transfer and move the files directly there.

————————————————————————————————————————————————————————————————————

Ios:

When downloading or airdropping the .mmp and .pd files, make sure to put both files inside the mobmuplat folder. 
You can do so by opening the Files app, going to download, selecting the files and the clicking on “Move”.

————————————————————————————————————————————————————————————————————
////////////////////////////////////////////////////////////////////

When updating the version of your .mmp and .pd files remember to overwrite or delete your previous version.
This has to be done to avoid conflicts between versions. The mobmuplat file is referenced to a specific pd file name, 
like “template.pd”, a file named “template (01).pd”[1]  won’t work. 


[1]This is exactly how Android renames the pd and mmp when you don’t delete the previous version.
